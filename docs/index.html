Guide d'installation

Comment connecter chaque service

Chaque plateforme se configure en quelques clics depuis le menu de GamePill.

🟣 Twitch

  1. Clic droit sur l'icône GamePill en bas à droite
  2. Twitch → Connecter
  3. Une page s'ouvre dans ton navigateur, accepte
  4. C'est tout. Follows, subs et raids arrivent en direct

🟢 Kick

  1. Menu → Kick → Configurer
  2. Entre le nom de ta chaîne Kick
    (celui qui apparaît dans l'URL : kick.com/tonnom)
  3. Tes viewers s'affichent automatiquement

🔴 YouTube

  1. Menu → YouTube → Connecter
  2. Entre ton nom de chaîne YouTube
  3. GamePill détecte si tu es en live et affiche tes viewers

⚔ Valorant & League of Legends

  1. Menu → Riot → Configurer
  2. Entre ton pseudo Riot au format Pseudo#TAG
  3. Sélectionne ta région
  4. Tes stats de la dernière partie s'affichent automatiquement

🔫 CS2, stats en direct

  1. Menu → CS2 → Installer le fichier de config
  2. Relance CS2
  3. Menu → CS2 → Activer
  4. Tes kills, HP et argent apparaissent en temps réel pendant la partie

⚙️ Autres réglages

  • Démarrage avec Windows → Menu → Démarrer avec Windows
  • Déplacer la pill → Clic droit et glisse
  • Changer d'écran → Glisse la pill sur l'écran voulu
